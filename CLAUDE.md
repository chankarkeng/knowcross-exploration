### Intro
This repo will be a wrapper to allow testing of Unifocus integration API and shall follow the security measure mentioned on @docs/_Unifocus Integration API -Operations.pdf to fulfil the authorization. This app should be written with minimal implementation just for API testing, design it in a way that each request and response should be logged to ease review as well.

Create self hosted swagger to ease the testing also 

Write this in Expressjs typescript to ease future integration to other BE services.
